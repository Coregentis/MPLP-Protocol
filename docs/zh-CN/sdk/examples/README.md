## 🚀 **快速开始**

### **运行示例**

所有示例都可以直接运行：

```bash
# 1. 克隆示例代码
git clone https://github.com/Coregentis/MPLP-Protocol.git
cd MPLP-Protocol/examples

# 2. 安装依赖
npm install

# 3. 运行示例
npm run example:simple-agent
```

## 💻 **代码结构**

每个示例都包含：

```
example-name/
├── src/
│   ├── index.ts          # 主入口
│   ├── Agent.ts          # Agent实现
│   └── config.ts         # 配置文件
├── tests/                # 测试文件
├── package.json          # 依赖配置
├── tsconfig.json         # TypeScript配置
```
