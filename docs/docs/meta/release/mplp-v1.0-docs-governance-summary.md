# Documentation Governance Summary



## 1. Overview

This document summarizes the governance structure for MPLP v1.0 documentation.

## 2. Governance Bodies

| Body | Role |
|:---|:---|
| **MPGC** | MPLP Protocol Governance Committee - final authority on protocol changes |
| **Maintainers** | Day-to-day repository management |
| **Contributors** | Community members submitting PRs |

## 3. Documentation Hierarchy

| Level | Scope | Change Process |
|:---|:---|:---|
| **Normative** | Schemas, Core Specs | MIP required |
| **Informative** | Guides, Examples | PR review |
| **Meta** | This document | Maintainer approval |
