# GF-02: Multi-Agent Coordination

> **Type**: Normative Validation Scenario
> **Profile**: MAP (Multi-Agent)
> **Status**: ✅ VERIFIED

## Scope

This golden flow validates:
- **Collab** session initialization
- **Role** binding for multiple participants
- **Turn-taking** logic between agents
- **Broadcast** communication patterns

## Non-Goals

This flow does NOT validate:
- Single-agent lifecycle (see GF-01)
- Drift detection (see GF-03)
- Governance confirmation (see GF-05)

## Normative Requirements

Implementations MUST:
- Ensure all participants have valid Role bindings
- Enforce turn-taking constraints defined in Collab mode
- Emit `MAPTurnDispatched` and `MAPTurnCompleted` events
- Maintain consistent shared state across all participants

---

## Expected Outcome

- All agents participate according to the Collab mode
- Messages are delivered in the correct order
- Session terminates gracefully with a final state snapshot
