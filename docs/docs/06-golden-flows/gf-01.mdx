# GF-01: Single-Agent Lifecycle

> **Type**: Normative Validation Scenario
> **Profile**: SA (Single Agent)
> **Status**: ✅ VERIFIED

## Scope

This golden flow validates:
- **Context** initialization from project configuration
- **Plan** creation with step decomposition
- **Trace** recording for each execution step
- **State transitions** are deterministic and immutable

## Non-Goals

This flow does NOT validate:
- Multi-agent coordination (see GF-02)
- Human-in-the-loop confirmation workflows (see GF-04)
- Drift detection and recovery (see GF-03)

---

## Modules Involved

| Module | Role in Flow |
|:-------|:-------------|
| **Context** | Provides initial constraints and objectives |
| **Plan** | Defines execution steps |
| **Trace** | Records all state transitions |
| **Confirm** | Validates governance at each step |

## Normative Requirements

Implementations MUST:
- Initialize Context before Plan creation
- Record all step transitions in Trace
- Ensure terminal Trace states are immutable
- Complete lifecycle without orphaned resources

---

## Expected Outcome

- Agent completes lifecycle without error
- All trace entries are immutable and ordered
- Final state matches expected output
