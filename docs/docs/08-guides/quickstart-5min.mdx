> [!IMPORTANT]
> **Informative, not normative.** This guide provides suggested steps, not protocol requirements.

## 1. Clone & Install

```bash
git clone https://github.com/Coregentis/MPLP-Protocol.git
cd MPLP-Protocol
pnpm install
```

---

## 2. Validate Protocol Schemas

```bash
pnpm test:schema
```

If the validation succeeds, your environment is MPLP-ready.

---

## 3. Run a Golden Flow

```bash
cd examples/ts-golden-flows
pnpm run flow:01
```

You should see a trace of MPLP envelopes moving through the lifecycle.
