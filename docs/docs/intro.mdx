## Adopt MPLP Incrementally

Recommended adoption path:

1. **Trace First** — Start with observability. Record what your agents do.  
   → [Trace Module](./02-modules/trace-module.md) · [Observability](./04-observability/observability-overview.md)

2. **Confirm Boundaries** — Add confirmation before high-risk actions.  
   → [Confirm Module](./02-modules/confirm-module.md) · [Governance](./12-governance/versioning-policy.md)

3. **Plan & Context** — Structure agent reasoning with explicit plans and shared context.  
   → [Plan Module](./02-modules/plan-module.md) · [Context Module](./02-modules/context-module.md)

4. **Profiles & Conformance** — Validate against execution profiles and golden flows.  
   → [SA Profile](./03-profiles/sa-profile.md) · [MAP Profile](./03-profiles/map-profile.md)
