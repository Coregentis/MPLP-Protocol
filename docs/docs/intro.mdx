# Documentation Overview

## Purpose of This Documentation

This documentation site serves as a **technical reference and implementation guide** for the MPLP (Multi-Agent Lifecycle Protocol).

It explains **how the protocol is structured**, **how it can be implemented**, and **how protocol behavior can be verified through self-declared conformance**.

This site is **not**:
- a product website
- a marketing or promotional surface
- a certification, endorsement, or compliance authority

For protocol positioning, governance boundaries, and standards relationships, refer to the canonical MPLP website at **https://mplp.io**.
