# Cross-Cutting Kernel Duties (11 OS-Level Duties)

> **Status**: Normative
> **Version**: 1.0.0
> **Authority**: MPGC
> **Protocol**: MPLP v1.0.0 (Frozen)

## Scope

This specification defines the 11 cross-cutting kernel duties that MUST be implemented by all MPLP-conformant runtimes. These duties span across all lifecycle phases and modules, providing the foundational behavioral contracts for runtime integration.

## Non-Goals

This specification does not mandate specific implementation details beyond the defined interfaces and invariants.
