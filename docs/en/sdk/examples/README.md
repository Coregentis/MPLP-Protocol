## 🚀 **Quick Start**

### **Running Examples**

All examples can be run directly:

```bash
# 1. Clone example code
git clone https://github.com/Coregentis/MPLP-Protocol.git
cd MPLP-Protocol/examples

# 2. Install dependencies
npm install

# 3. Run example
npm run example:simple-agent
```
