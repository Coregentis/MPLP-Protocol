## üéØ Enterprise Quick Start

### **1. Environment Setup**
```bash
# Install MPLP Alpha version
npm install mplp@alpha

# Verify installation
npm list mplp
# Should display: mplp@1.0.0-alpha
```

### **2. Basic Configuration**
```typescript
import { MPLPCore } from 'mplp';

const mplp = new MPLPCore({
  version: '1.0.0-alpha',
  modules: {
    // Enable all 10 completed modules
    context: true,
    plan: true,
    role: true,
    confirm: true,
    trace: true,
    extension: true,
    dialog: true,
    collab: true,
    core: true,
    network: true
  }
});

await mplp.initialize();
console.log('üöÄ MPLP v1.0 Alpha initialized');
```
