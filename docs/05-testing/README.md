## 🔍 测试方法论

### Schema驱动测试原则
1. **基于实际实现**: 所有测试基于真实的TypeScript接口和Schema
2. **避免假设性测试**: 不使用假设的接口或数据结构
3. **类型安全保证**: 确保测试代码与生产代码类型一致
4. **接口一致性**: 验证测试接口与实际实现的完全匹配

### 三层测试策略
1. **单元测试**: 快速反馈，隔离测试，高覆盖率
2. **集成测试**: 模块协作，接口验证，数据流测试
3. **端到端测试**: 业务流程，用户场景，性能验证
