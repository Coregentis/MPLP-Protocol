- **Role-based**: Automatic approval for certain roles
- **Value-based**: Approval based on request values

#### Rule Engine
```typescript

  enabled: boolean;
  conditions: ApprovalCondition[];
  action: 'approve' | 'reject' | 'escalate';
  priority: number;
}

  field: string;
  operator: 'equals' | 'greater_than' | 'less_than' | 'contains';

```
